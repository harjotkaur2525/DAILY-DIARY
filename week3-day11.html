K-Nearest Neighbors & Confusion Matrix

K-Nearest Neighbors (KNN) Algorithm

Introduction: KNN is a supervised learning algorithm used for both classification and regression. It classifies a data point based on how its neighbors are classified.

Working:

  • Choose the number of neighbors (k)
  • Calculate the distance (e.g., Euclidean) from the query point to the labeled points
  • Pick the k closest neighbors
  • Return the most common class (for classification) or average (for regression)

Uses:

  • Recommender systems
  • Image recognition
  • Medical diagnostics

Example (Python):

from sklearn.neighbors import KNeighborsClassifier
from sklearn.datasets import load_iris

iris = load_iris()
X, y = iris.data, iris.target

model = KNeighborsClassifier(n_neighbors=3)
model.fit(X, y)

prediction = model.predict([[5.1, 3.5, 1.4, 0.2]])
print("Predicted class:", prediction)

Confusion Matrix

Introduction: A confusion matrix is a performance measurement for classification problems. It shows the actual vs predicted classifications.

Parameters:

  • True Positive (TP): Correctly predicted positive cases
  • True Negative (TN): Correctly predicted negative cases
  • False Positive (FP): Incorrectly predicted positive cases
  • False Negative (FN): Incorrectly predicted negative cases

Example (Python):

from sklearn.metrics import confusion_matrix

y_true = [0, 1, 1, 0, 1, 1, 0]
y_pred = [0, 1, 0, 0, 1, 1, 1]

cm = confusion_matrix(y_true, y_pred)
print("Confusion Matrix:\n", cm)
