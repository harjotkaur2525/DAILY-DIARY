Week 2 - Day 4

Hands-on Practice & Mini Project

Hands-on Practice

  • Building Project on Wokwi Simulator: Students will create a basic IoT circuit using ESP32 and DHT sensors in the virtual environment.
  • Uploading Data to Cloud (ThingSpeak): Using HTTP API or ThingSpeak libraries to push sensor readings for real-time visualization.

Mini Project Overview

  • Objective: Read Temperature and Humidity data using DHT11 or DHT22.
  • Board Used: ESP32 Development Board
  • Steps:
    • Connect DHT11/DHT22 sensor to ESP32
    • Write code to read sensor data
    • Send data to ThingSpeak channel using Wi-Fi
    • Visualize data using ThingSpeak graphs
