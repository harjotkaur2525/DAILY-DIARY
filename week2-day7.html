Data Visualization

Visualizing Data Using Graphs

Visualization helps understand patterns, trends, and outliers in data. Common types of graphs include:

  • Bar Graph: Shows categorical data with rectangular bars.
  • Line Graph: Displays data points connected by straight lines, useful for trends.
  • Histogram: Represents the distribution of numerical data.
  • Scatter Plot: Shows relationships between two variables.
  • Box Plot: Displays the distribution and identifies outliers.
  • Heat Map: Uses color to represent values in a matrix format.

Libraries Used

  • Matplotlib: A powerful library for creating static, animated, and interactive visualizations in Python.
    • Commonly used module: pyplot
  • Seaborn: Built on top of matplotlib, it provides a high-level interface for attractive and informative graphics.

Example Dataset

The Tips Dataset is a common dataset used to demonstrate data visualization. It contains information about tips received by waiters and includes features such as total bill, tip amount, gender, smoking status, day, and time.

Real-Time Data Plotting

Real-time plotting allows monitoring of changing data over time, which is useful in IoT and sensor applications.

Python libraries used include:

  • Matplotlib: Can be used for live plotting using the animation module.
  • Random: Generates random data to simulate sensor input.
  • Time: Used to add delay or timestamps between data points.
