Computing Paradigms, Cloud & ThingSpeak

Introduction to Computing and its Types

  • Traditional Computing: Involves centralized processing, limited scalability, and local data storage.
  • Distributed Computing: Breaks tasks across multiple machines. Includes:
    • Cluster Computing: Group of connected computers acting as one system.
    • Grid Computing: Connects geographically distributed computers to achieve a common goal.
  • Edge Computing: Processes data near the source/device, reducing latency and bandwidth use.
  • Quantum Computing: Uses quantum bits for complex computations far beyond classical limits.
  • Cloud Computing: Internet-based computing for on-demand resources:
    • IaaS: Infrastructure as a Service (e.g., virtual machines).
    • PaaS: Platform as a Service (e.g., app development platforms).
    • SaaS: Software as a Service (e.g., Gmail, Office 365).

What is Virtualization

Virtualization allows multiple operating systems or applications to run on a single physical machine using virtual machines. It enhances resource utilization and system efficiency.

Cloud Computing in IoT

  • Why it is needed: Enables scalable storage, real-time analytics, and remote access to IoT data.
  • Cloud Platforms: AWS IoT, Google Cloud IoT, Azure IoT offer device management, analytics, and security features.

IoT Cloud Architecture

Includes device layer, gateway layer, cloud services (data ingestion, processing, analytics), and application interfaces for end-users. Ensures seamless data flow and control.

Introduction to ThingSpeak

ThingSpeak is an open-source IoT platform that enables data collection, visualization, and analysis using channels and MATLAB-based analytics. Ideal for prototyping and academic projects.
