Exploratory Data Analysis & Mini Project Planning

Introduction to EDA (Exploratory Data Analysis)

  • What it does: Helps summarize the main characteristics of data using visual methods.
  • Need: Essential for understanding dataset patterns, spotting anomalies, and making decisions on data preprocessing and modeling.

Gathering Information for Mini Project 2

Identify the sensor, microcontroller, and cloud platform needed. Determine the type of data to be collected and the purpose of the dashboard.

Creating Flowchart for the Project

Design a clear flow of how data will move from sensors to the dashboard including collection, transmission, processing, and visualization.

Mini Project 2 - Real-Time IoT Data Dashboard

  • Collect real-time sensor data using ESP32 or Arduino.
  • Send the data to ThingSpeak or similar platform.
  • Create live graphs to visualize temperature, humidity, or other data.
  • Display data in a clean and dynamic dashboard interface.
